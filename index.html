Calibre View

Dropbox Access & App Status

Calibre View uses the Dropbox API to let you browse and download books from your personal Calibre library stored in Dropbox.

At launch, Calibre View operates under Dropbox’s development status, which allows up to 500 linked users. This comfortably supports early releases, TestFlight users, and initial public availability.

Dropbox requires apps to apply for production status once they reach 50 linked users. At that point, the app must be submitted for review within two weeks. Until approval is granted, no additional users beyond that threshold can link their accounts.

If Calibre View grows quickly, an early review may be requested to ensure uninterrupted access as the app scales.

Calibre View follows Dropbox’s API usage and branding guidelines. The app only accesses files explicitly selected by the user and never modifies, uploads, or shares content without user action.
